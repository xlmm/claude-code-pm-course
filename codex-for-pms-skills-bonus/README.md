# Codex for PMs — Skills Module

Five interactive lessons that teach you to build, use, and share Codex skills — taken from the Full Stack PM Mastery course and adapted for the Codex desktop app.

## Setup

1. Download this folder (or clone the repo) anywhere on your computer.
2. Open the **Codex desktop app**.
3. Click **Work in a project** → **Add new project** → **Use an existing folder**, and pick this folder.
4. In the message box, type:

```
/start skills 1
```

That's it. The lesson takes it from there.

## The lessons

| Command | Lesson |
|---|---|
| `/start skills 1` | Your First Skill — build a working skill from scratch |
| `/start skills 2` | Types of Skills — tour the five skill types with live demos |
| `/start skills 3` | Advanced Techniques — folders, templates, skills that ask, composition |
| `/start skills 4` | Philosophy of Skills — when to build one, and how to keep them sharp |
| `/start skills 5` | The Ecosystem — installing, sharing, and the open skills standard |

Lessons build on each other — start with 1 even if you've built skills before (it'll offer you a skip if you're already comfortable).

## What's in this folder

- `.agents/skills/start-skills-*` — the lesson scripts
- `.agents/skills/` (everything else) — real, working PM skills the lessons demo: meeting processing, proposal writing, doc conversion, follow-up drafting, and more. They're yours to keep and adapt.
- `meetings/` appears after Lesson 1 starts — example transcripts the lessons work with.
