# Proposal Template

## Executive Summary
2-4 sentences. What are we doing, why, and what's the expected impact.

## Problem
What's the current situation? What's broken or missing? Include data or evidence if available.

## Proposed Solution
What specifically are we building or changing? Be concrete — name the deliverables.

## Scope
What's included and what's explicitly NOT included. Define the boundaries early.

## Timeline
Phase breakdown with rough dates. Include key milestones and decision points.

## Resources Needed
People, tools, budget. Be specific enough to get approval.

## Success Metrics
How will we know this worked? 2-3 measurable outcomes.

## Risks and Mitigations
What could go wrong? What's the plan if it does?

## Next Steps
The first 3 things that need to happen if this is approved.
