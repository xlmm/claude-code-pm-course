# Claude Code for Product Managers - Course Materials

Welcome to the interactive Claude Code course for Product Managers!

## Getting Started

**You're in the right place!** This folder contains everything you need to complete the course. It was installed by the FSPM CLI (`fspm get claude-code-for-pms`).

### What's Included

- **`lesson-modules/`** - Interactive modules taught by Claude
- **`company-context/`** - TaskFlow company reference materials for exercises
- **`.claude/`** - Slash commands that power the course experience

### How to Start the Course

**Claude Desktop app:** Open the project where this course was installed and type:

```
/start-1-1
```

**Claude Code CLI:** Open your terminal in this folder, launch Claude Code, then start the first module:

```bash
claude
```

Once Claude Code is running, type:

```
/start-1-1
```

This kicks off Module 1.1 - Welcome to TaskFlow!

### Course Structure

**Module 0: Getting Started** (Prerequisites)
- Before running this course, you should have Claude installed
- Setup guide: https://ccforpms.com/getting-started/installation

**Module 1: Fundamentals** (Starts with `/start-1-1`)

**Module 2: Advanced PM Work**
- 2.1: Write a PRD

- 2.3: Product Strategy

**Module 3: Nano Banana** (Starts with `/start-3-1-1`)
- AI image generation for PM work with Gemini

**Module 4: Vibe Coding** (Starts with `/start-4-1`)
- Build and deploy a real web app

### Slash Commands

Throughout the course, you'll use slash commands like:
- `/start-1-1` - Start Module 1.1
- `/start-2-1` - Start Module 2.1
- etc.

These commands are defined in the `.claude/` folder and guide you through each module interactively.
