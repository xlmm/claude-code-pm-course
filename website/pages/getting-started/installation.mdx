## 0.1: Installation

 - **Time to Complete:** 5 minutes
 - **Prerequisites:** Claude Pro or Max subscription

## Overview

This course runs in the **Claude Desktop app**. Install the app, sign in, paste one prompt, and Claude sets up the whole course for you.

*Prefer the terminal? → [Claude Code CLI guide](/getting-started/claude-code-cli)*

**Key takeaway:** You need a paid Claude subscription and the Claude Desktop app. Setup takes about 5 minutes.


## Prerequisites

| Requirement | Details |
|-------------|---------|
| **Claude Subscription** | Claude Pro ($20/month) or Claude Max. Free tier NOT supported. Sign up at [claude.ai](https://claude.ai/) |
| **Operating System** | macOS or Windows 10+ (Linux users: use the [Claude Code CLI guide](/getting-started/claude-code-cli)) |


## Install the Claude Desktop App

**Mac:**
1. Download the Mac app from [claude.ai/download](https://claude.ai/download)
2. Open the downloaded file and drag Claude into your Applications folder
3. Launch Claude and sign in with your Pro or Max account

**Windows:**
1. Download the Windows installer from [claude.ai/download](https://claude.ai/download)
2. Run the installer and follow the steps
3. Launch Claude and sign in with your Pro or Max account

That's it. No terminal, no Node.js, no extra tools.

### Verify It Works

Open Claude and start a new chat. If Claude responds, you're set.
