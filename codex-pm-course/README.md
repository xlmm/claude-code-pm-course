# Codex for Product Managers

An interactive course that teaches product managers how to use Codex for practical PM work.

**Course site:** https://codexforpms.com

## How To Start

1. In the Codex app, start a new chat and click **Work in a project** → **Add new project** → **Use an existing folder**, then select this folder.
2. Start the first lesson — type:

```text
/start 1 1
```

3. Follow the instructor prompts.

## Course Structure

The course is packaged as Codex skills in `.agents/skills/`. Each lesson is self-contained: the teaching script and every file needed for that lesson live inside that lesson's skill folder.

Start with `/start 1 1` and continue through the lesson prompts.
