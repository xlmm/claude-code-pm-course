# Project Memory Reference

Use `AGENTS.md` to capture lean, durable project context that Codex should load when working in a project.

Good `AGENTS.md` files are usually maps, not encyclopedias. They explain where important files live, which conventions are stable, and how to approach work in the repo.

Subfolder `AGENTS.md` files can provide narrower guidance for a specific area, such as `user-interviews/AGENTS.md` for research synthesis.
