# TaskFlow Product Context

TaskFlow helps teams organize work into Workspaces, Tasks, and Epics.

## Product Terms

- Workspace: the main container for a team's work
- Task: an individual unit of work
- Epic: a larger body of work made up of related Tasks

## Current Product Focus

The activation team is improving the path from signup to first useful Workspace. The main goal is to reduce confusion and help teams reach value faster.
